# QA Testing Report – Discord

## Description
This project showcases manual testing performed on the Discord application. It includes the creation of structured test cases and documented bug reports based on real testing scenarios.

## Scope
- Manual testing of Discord screen sharing feature
- Identification of intermittent bugs
- Validation through different test scenarios

## Tools used
- Manual testing
- Test Case Design
- Bug Reporting

## Contents
- Test Cases
- Bug Reports
