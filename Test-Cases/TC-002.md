# Verify screen sharing display after rejoining the call

## ID
TC-002

## Prerequisites
- User A and User B are in a call
- User A has permission to share their screen
- User B is able to leave and rejoin the call

## Steps
1. User A starts screen sharing
2. User B joins the call
3. User B leaves the call
4. User B rejoins the call
5. Observe the interface

## Expected result
- The screen sharing preview is displayed
- The “Join” button is visible
- The stream is accessible after rejoining
