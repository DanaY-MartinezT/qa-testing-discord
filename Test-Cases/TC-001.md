# Verify that the screen sharing works correctly across different devices (laptop and smartphone)

## ID
TC-001

## Prerequisites
- User A and User B are in a call
- User A has permission to share screen from a mobile device
- User B will view the stream from a laptop

## Steps
1. User A starts screen sharing from a mobile device
2. User B joins the call from a laptop
3. User B checks if the screen sharing is visible

## Expected result
- The screen sharing preview is displayed
- The “Join” button is visible
- The stream can be accessed without issues
