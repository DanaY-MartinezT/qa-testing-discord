# Screen sharing is not displayed when starting a call

## ID
BUG-001

## Priority
Medium

## Severity
High

## Frequency
Occasional / Intermittent

## Description
When starting a screen share during a call, the interface does not display the preview or any visual indicator, although the start sound is played. This issue does not occur consistently.

## Steps to Reproduce
1. Start a call on Discord
2. Ask another user to start screen sharing
3. Repeat the process multiple times if necessary
4. Observe the receiver’s interface

## Expected Result
The screen sharin indicator should appear, and the user should be able to view or join the stream.

## Actual Result
No visual indicator or preview appears, even though the start sound is heard.

## Workaround
Leave the call and rejoin it.
